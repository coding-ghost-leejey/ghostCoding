네플릭스에서 어떤 콘텐츠를 시청할 수 있나요?

넷플릭스는 장편 영화, 다큐멘터리, 시리즈, 애니메이션, 각종 상을 수상한 넷플릭스 오리지널 등 수많은 콘텐츠를 확보하고 있습니다. 마음에 드는 콘텐츠를 원하는 시간에 원하는 만큼 시청하실 수 있습니다.

HTML 기본 태그 보기

주요 HTML 태그들:

  • <html> - HTML 문서의 루트 요소
  • <head> - 문서의 메타데이터
  • <body> - 문서의 본문 내용
  • <div> - 블록 레벨 컨테이너
  • <span> - 인라인 컨테이너

자주 사용하는 CSS 속성:

  • color - 텍스트 색상
  • background-color - 배경 색상
  • font-size - 글자 크기
  • font-weight - 글자 굵기

📚 datalist와 details 태그 예제

datalist는 input 요소에 미리 정의된 옵션 목록을 제공하여 사용자가 쉽게 선택할 수 있게 해줍니다.

details는 사용자가 클릭하여 펼치거나 접을 수 있는 콘텐츠를 만들어줍니다.

📝 datalist 태그 예제

datalist는 input 요소와 함께 사용되어 자동완성 기능을 제공합니다.




🔄 option 태그 작성법 비교

방법 1: <option value="HTML"> (현재 사용 중)

방법 2: <option>HTML</option>



💡 결론: 두 방법 모두 동일하게 작동합니다! 하지만 value 속성을 사용하는 것이 더 명시적이고 확장성이 좋습니다.

⚠️ select 태그와 datalist의 차이점

중요: select 태그에서는 option에 반드시 텍스트 내용이 필요합니다!



🚨 주의사항:
  • Select: <option value="html">HTML</option> (텍스트 내용 필수!)
  • Datalist: <option value="HTML"> 또는 <option>HTML</option> (둘 다 가능)

📋 details 태그 예제

details는 사용자가 클릭하여 펼치거나 접을 수 있는 콘텐츠를 만듭니다.

📚 HTML 기본 태그 보기

주요 HTML 태그들:

  • <html> - HTML 문서의 루트 요소
  • <head> - 문서의 메타데이터
  • <body> - 문서의 본문 내용
  • <div> - 블록 레벨 컨테이너
  • <span> - 인라인 컨테이너
  • <p> - 단락
  • <h1> ~ <h6> - 제목
  • <ul>, <ol> - 목록
  • <img> - 이미지
  • <a> - 링크
🎨 CSS 속성 보기

자주 사용하는 CSS 속성:

  • color - 텍스트 색상
  • background-color - 배경 색상
  • font-size - 글자 크기
  • font-weight - 글자 굵기
  • margin - 바깥 여백
  • padding - 안쪽 여백
  • border - 테두리
  • border-radius - 모서리 둥글게
  • display - 표시 방식
  • position - 위치 지정
⚙️ JavaScript 메서드 보기

유용한 JavaScript 메서드:

  • querySelector() - DOM 요소 선택
  • querySelectorAll() - 여러 DOM 요소 선택
  • addEventListener() - 이벤트 리스너 추가
  • innerHTML - HTML 내용 변경
  • textContent - 텍스트 내용 변경
  • classList.add() - CSS 클래스 추가
  • classList.remove() - CSS 클래스 제거
  • localStorage.setItem() - 로컬 저장소에 저장
  • localStorage.getItem() - 로컬 저장소에서 가져오기
  • fetch() - API 호출
🔍 열린 상태의 details (open 속성 사용)

이 details는 open 속성이 설정되어 있어서 페이지 로드 시 자동으로 열려있습니다.

사용자가 원할 때 접을 수 있습니다.

사용법: <details open>

💡 사용 팁

  • datalist: 사용자가 자유롭게 입력할 수 있으면서도 제안된 옵션을 제공할 때 유용합니다.
  • details: 긴 설명이나 추가 정보를 숨겨두고 필요할 때만 보여주고 싶을 때 사용합니다.
  • datalist의 option은 value 속성만 필요하며, 텍스트 내용은 자동으로 value 값이 됩니다.
  • details는 접기/펼치기 상태를 JavaScript로 제어할 수도 있습니다.
